# bash-script 

Набор учебных Bash-скриптов для отработки основ автоматизации в терминале.

## Список скриптов

- `greet.sh` — запрашивает имя и приветствует пользователя.
- `sum.sh` — складывает два числа.
- `even_odd.sh` — проверяет, чётное ли число.
- `project_init.sh` — создаёт базовую структуру веб-проекта.
- `line_counter.sh` — считает строки в указанном файле.
- `password_gen.sh` — генерирует случайный пароль из 8 символов.
- `find_by_ext.sh` — ищет файлы по расширению в текущей директории.

## Как использовать

1. Клонируйте репозиторий.
2. Сделайте скрипты исполняемыми:
   ```bash
   chmod +x *.sh

Запускайте любой скрипт:

   ./script_name.sh
